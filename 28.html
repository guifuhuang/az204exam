Azure AD认证测试题

QUESTION NO: 28 - 企业网站用户认证

问题描述

您需要根据架构图对企业网站进行用户认证。

应该使用哪两个值？每个正确答案都提供了解决方案的一部分。

注意：每个正确选项值1分。

提示：本题为多选题，请选择两个正确答案。根据Azure AD认证流程，用户认证需要特定的标识和终结点信息。
正确答案与解析

正确答案：A 和 D

A. ID token signature (ID令牌签名)
D. Azure AD endpoint URI (Azure AD终结点URI)

解析说明：

根据Azure AD认证架构和文档内容：

Header
算法和令牌类型
Payload
声明(Claims)
Signature
验证令牌

1. ID token signature (ID令牌签名)：

  • 用于验证令牌的完整性和真实性
  • 确保令牌未被篡改
  • 是JWT(JSON Web Tokens)的三个组成部分之一

2. Azure AD endpoint URI (Azure AD终结点URI)：

  • 是用户进行认证的入口点
  • 客户端可以从v1.0或v2.0端点获取访问令牌
  • 浏览器将用户重定向到此终结点进行登录

其他选项分析：

  • B. ID token claims：包含用户身份信息，但签名用于验证这些信息的真实性
  • C. HTTP response code：表示请求状态，不用于认证本身
  • E. Azure AD tenant ID：是租户标识符，已包含在终结点URI中
