QUESTION NO: 317

Your company has several containers based on the following operating systems:

  • Windows Server 2019 Nano Server
  • Windows Server 2019 Server Core
  • Windows Server 2022 Nano Server
  • Windows Server 2022 Server Core
  • Linux

You plan to migrate the containers to an Azure Kubernetes cluster. What is the minimum number of node pools that the cluster must have?

正确答案: B. 2

说明:

Azure Kubernetes Service (AKS) 集群需要至少两个节点池来支持不同的操作系统类型：

  1. 一个用于 Linux 容器的节点池
  2. 一个用于 Windows 容器的节点池（可以同时支持 Windows Server 2019 和 2022 的不同版本）

Linux 和 Windows 容器不能运行在同一个节点池中，因为它们需要不同的操作系统基础镜像。虽然 Windows Server 2019 和 2022 有不同的版本（Nano Server 和 Server Core），但它们可以共存于同一个 Windows 节点池中。

因此，最小需要的节点池数量是 2 个：一个 Linux 节点池和一个 Windows 节点池。
