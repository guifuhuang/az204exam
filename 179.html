Margie's Travel is an international travel and bookings management service. The company is expanding into restaurant bookings. You are tasked with implementing Azure Search for the restaurants listed in their solution.

You create the index in Azure Search.

You need to import the restaurant data into the Azure Search service by using the Azure Search .NET SDK.

Solution:

  1. Create a SearchServiceClient object to connect to the search index.
  2. Create a DataContainer that contains the documents which must be added.
  3. Create a DataSource instance and set its Container property to the DataContainer.
  4. Set the DataSources property of the SearchServiceClient.

Does the solution meet the goal?

正确答案：B. No

说明：

根据Azure Search的.NET SDK使用方式，正确的步骤应该是：

  1. 创建SearchIndexClient或SearchClient来连接搜索索引
  2. 准备要索引的文档数据
  3. 使用Documents.Index或IndexAsync方法导入数据

Use the following method:

  1. - Create a SearchIndexClient object to connect to the search index
  2. - Create an IndexBatch that contains the documents which must be added.
  3. - Call the Documents.Index method of the SearchIndexClient and pass the IndexBatch

题目中提到的解决方案有几个问题：

  • SearchServiceClient是旧版SDK中的类，新版使用SearchIndexClient/SearchClient
  • 数据导入应该使用Documents属性上的操作方法，而不是通过DataSource设置
  • DataContainer和DataSource概念在数据导入流程中不适用
