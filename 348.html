QUESTION NO: 348 HOTSPOT

You develop and deploy an Azure App Service web app that connects to Azure Cache for Redis as a content cache. All resources have been deployed to East US 2 region.

The security team requires the following from Azure Cache for Redis:

  • The number of Redis client connections from an associated IP address
  • Redis operations completed on the content cache
  • The location (region) in which the Azure Cache for Redis instance was accessed

The audit information must be captured and analyzed by a security team application deployed to Central US region.

You need to log information on all client connections to the cache.

Which configuration values should you use?

Requirement Configuration value
Store log information
Enable client connection logging

正确答案:

Store log information: Log Analytics workspace

Enable client connection logging: Managed identity

解释说明:

1. Log Analytics workspace 是存储日志信息的最佳选择。它可以收集和分析来自不同Azure资源的日志数据，支持跨区域查询，非常适合安全团队分析Redis缓存访问情况。

2. Managed identity 是启用客户端连接日志记录的正确配置。托管身份为Azure资源提供自动管理的身份，可以安全地访问其他Azure资源（如Log Analytics工作区），无需管理凭据。

这些选择是基于以下原因：

  • Log Analytics workspace支持跨区域日志收集和分析
  • Managed identity提供安全的身份验证机制，确保日志数据的安全传输
  • 组合使用这两种配置可以安全地记录和分析客户端连接信息
  • Diagnostic setting虽然可以收集日志，但需要配合身份验证机制使用
  • Managed identity是Azure推荐的安全实践，用于资源间的安全通信
