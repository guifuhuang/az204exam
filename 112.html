QUESTION NO: 112 DRAG DROP

You manage several existing Logic Apps.

You need to change definitions, add new logic, and optimize these apps on a regular basis.

What should you use? To answer, drag the appropriate tools to the correct functionalities. Each tool may be used once, more than once, or not at all. You may need to drag the split bar between panes or scroll to view content.

NOTE: Each correct selection is worth one point.

Tools

Logic Apps Designer
Code View Editor
Enterprise Integration Pack

Functionality

Edit B2B workflows
Edit definitions in JSON
Visually add functionality

正确答案:

Edit B2B workflows: Enterprise Integration Pack

Edit definitions in JSON: Code View Editor

Visually add functionality: Logic Apps Designer

说明:

根据Azure Logic Apps的功能和题目要求，以下是正确的工具匹配：

详细解释:

1. Edit B2B workflows: Enterprise Integration Pack

Enterprise Integration Pack (EIP) 专门用于处理B2B工作流，支持EDI、X12、AS2等企业集成场景。

2. Edit definitions in JSON: Code View Editor

Code View Editor 允许直接编辑Logic Apps的JSON定义文件，适合进行底层配置修改。

3. Visually add functionality: Logic Apps Designer

Logic Apps Designer 提供可视化界面，通过拖拽方式添加新的触发器和操作。

工具对比:

工具 主要用途 适用场景
Logic Apps Designer 可视化设计 添加新逻辑、修改流程结构
Code View Editor 代码编辑 修改定义、优化配置
Enterprise Integration Pack 企业集成 B2B工作流、EDI处理
