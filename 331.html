QUESTION NO: 331 HOTSPOT

You develop and deploy a web app to Azure App service. The web app allows users to authenticate by using social identity providers through the Azure B2C service. All user profile information is stored in Azure B2C.

You must update the web app to display common user properties from Azure B2C to include the following information:

  • Email address
  • Job title
  • First name
  • Last name
  • Office Location

You need to implement the user properties in the web app.

Requirement Value
API to access user properties
Code library to interface to Azure AD B2C

正确答案:

API to access user properties: Azure AD Graph

Code library to interface to Azure AD B2C: Azure Identity Library

解释说明:

1. Azure AD Graph 是访问用户属性的正确API选择。虽然Microsoft Graph是较新的API，但在某些Azure AD B2C场景中，Azure AD Graph仍然是推荐的选择，特别是在需要访问特定用户属性时。

2. Azure Identity Library 是与Azure AD B2C交互的正确代码库。Azure Identity Library提供了与Azure AD（包括B2C）集成的简化方式，支持获取访问令牌和管理身份验证。

这些选择是基于以下原因：

  • Azure AD Graph在某些Azure AD B2C实现中仍然是官方推荐的选择
  • Azure Identity Library专门为Azure服务身份验证设计，提供了与Azure AD B2C集成的简化方式
  • 其他选项如MSAL更适合Microsoft Graph API的交互，而不是Azure AD Graph
