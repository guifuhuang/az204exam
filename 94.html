Azure NoSQL Database Connection Test

QUESTION NO: 94

You develop Azure solutions.

You must connect to a No-SQL globally-distributed database by using the .NET API.

You need to create an object to configure and execute requests in the database.

Which code segment should you use?

答案及说明

正确答案: C. new CosmosClient(EndpointUri, PrimaryKey);

说明:

在Azure Cosmos DB的.NET SDK中：

  • CosmosClient 是用于配置和执行数据库请求的主入口点
  • Container和Database对象是通过CosmosClient实例创建的，不能直接实例化
  • 正确的连接流程是：先创建CosmosClient，然后通过它获取Database和Container实例
  • EndpointUri和PrimaryKey是连接Cosmos DB服务所需的认证信息

因此，选项C是唯一正确的选择，它创建了配置和执行请求所需的基础客户端对象。
