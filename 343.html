QUESTION NO: 343 HOTSPOT

You need to implement the Azure Function for delivery driver profile information.

Which configurations should you use? To answer, select the appropriate options in the answer area.

NOTE: Each correct selection is worth one point.

Configuration Value
Code library
API

正确答案:

Code library: Microsoft Authentication Library (MSAL)

API: Microsoft Graph

解释说明:

1. Microsoft Authentication Library (MSAL) 是用于实现交付司机个人信息功能的最佳代码库选择。MSAL是微软推荐的用于处理身份验证的库，支持多种身份验证流程，并专门针对Azure AD进行了优化。

2. Microsoft Graph 是访问用户个人信息的最佳API选择。它可以获取用户配置文件信息，包括司机相关的个人信息，是微软推荐的统一API端点。

这些选择是基于以下原因：

  • MSAL支持最新的身份验证协议和最佳实践
  • Microsoft Graph提供了统一的API来访问用户数据，取代了旧的Azure AD Graph API
  • MSAL与Microsoft Graph API配合使用是最佳实践
  • 其他选项如Key Vault SDK/API更适合机密管理而非用户数据访问
  • Azure Identity library更适合服务到服务的身份验证场景
