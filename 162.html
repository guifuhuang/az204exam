QUESTION NO: 162 DRAG DROP

You are developing a Docker/Go using Azure App Service Web App for Containers. You plan to run the container in an App Service on Linux. You identify a Docker container image to use.

None of your current resource groups reside in a location that supports Linux. You must minimize the number of resource groups required.

You need to create the application and perform an initial deployment.

Which three Azure CLI commands should you use to develop the solution? To answer, move the appropriate commands from the list of commands to the answer area and arrange them in the correct order.

Azure CLI Commands

az group create
az group update
az webapp update
az webapp create
az appservice plan create

Answer Area

⇦⇦
⇩⇩

正确答案:

  1. az group create
  2. az appservice plan create
  3. az webapp create

说明:

要在支持Linux的区域创建并部署容器化应用，正确的Azure CLI命令顺序应为：

  1. az group create - 首先在支持Linux的区域创建新的资源组
  2. az appservice plan create - 创建支持Linux的App Service计划
  3. az webapp create - 创建Web应用并配置容器部署

注意：

  • 题目要求最小化资源组数量，因此需要创建新的资源组而不是更新现有资源组
  • Linux支持需要在创建资源组和App Service计划时就指定
  • 不需要使用az webapp update或az group update命令
