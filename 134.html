QUESTION NO: 134

You provide an Azure API Management managed web service to clients. The back end web service implements HTTP Strict Transport Security (HSTS).

Every request to the backend service must include a valid HTTP authorization header.

You need to configure the Azure API Management instance with an authentication policy.

Which two policies can you use? Each correct answer presents a complete solution.

NOTE: Each correct selection is worth one point.

正确答案：

B. Basic Authentication

C. OAuth Client Credential Grant

说明：

  • Basic Authentication (B)：
    • 直接在HTTP头中添加Authorization头（Base64编码的凭据）
    • 符合题目要求的"valid HTTP authorization header"
    • API Management原生支持Basic认证策略
  • OAuth Client Credential Grant (C)：
    • 生成包含Bearer token的Authorization头
    • 符合HTTP授权头的规范要求
    • 适合服务到服务的认证场景
  • 排除选项：
    • Certificate Authentication (A)：使用TLS客户端证书，不会生成HTTP Authorization头
    • Digest Authentication (D)：虽然技术上可行，但API Management策略中不直接支持Digest认证

题目明确要求必须包含HTTP authorization header，只有Basic和OAuth两种方式会生成标准的Authorization头。根据Azure API Management官方文档，这两种是推荐的后端认证方式。
