Microsoft Identity Platform - Token Request

QUESTION NO: 357
You are developing a web application that uses the Microsoft identity platform to authenticate users and resources. The web application calls several REST APIs. The APIs require an access token from the Microsoft identity platform. You need to request a token. Which three properties should you use? Each correct answer presents part of the solution.
NOTE: Each correct selection is worth one point.

答案及说明

正确答案: B, C, E (Application secret, Application ID, Redirect URI/URL)

说明:

根据文档内容，在请求访问令牌时需要以下三个属性：

  1. Application secret (B)：客户端密码，用于验证应用程序的身份（文档中多次提到在令牌请求中使用client_secret）。
  2. Application ID (C)：应用程序的唯一标识符，在注册应用程序时分配（文档中称为"应用程序ID"或"client_id"）。
  3. Redirect URI/URL (E)：授权服务器完成身份验证后重定向用户的终结点（文档中明确说明重定向URI是必需配置项）。

其他选项不正确的原因：

  • Application name (A)：应用程序名称是描述性信息，在令牌请求中不需要。
  • Supported account type (D)：支持的帐户类型是在应用程序注册时配置的，但不用于令牌请求过程。

文档参考：在《azure-active-directory-b2c.pdf》中明确说明了注册Web应用程序时需要配置Application ID和Redirect URI，并在获取令牌时需要提供Application secret。
