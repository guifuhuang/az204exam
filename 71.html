Azure迁移项目 - 内部员工访问控制测试题

QUESTION NO: 71
HOTSPOT

You need to provide internal staff access to the production site after a validation.

How should you complete the code segment? To answer, select the appropriate options in the answer area.

NOTE: Each correct selection is worth one point.

<a href="https://www.munsonspicklesandpreservesfarm.com/?______=______">Go back to production app</a>

正确答案与解析

正确选项:
下拉框1: x-ms-routing-name
下拉框2: production

解析说明

在Azure部署环境中，x-ms-routing-name是一个特殊的查询参数，用于控制请求路由到哪个部署槽(slot)。

根据Munson's Pickles and Preserves Farm的需求：

  • 内部员工在验证后需要访问生产环境(production)
  • 文档要求"提供直接链接回到生产应用"
  • 使用x-ms-routing-name=production明确指定路由到生产槽

其他选项解析：

  • x-ms-app: 用于标识应用程序，不控制路由
  • x-ms-user: 用于传递用户信息，与路由无关
  • self: 表示当前槽位，在验证环境中可能指向临时槽
  • staging: 表示过渡环境，不符合要求

因此，正确的代码应该是：

<a href="https://www.munsonspicklesandpreservesfarm.com/?x-ms-routing-name=production">Go back to production app</a>
