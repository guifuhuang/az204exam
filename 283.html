QUESTION NO: 283

You are creating an app that will use CosmosDB for data storage. The app will process batches of relational data.

You need to select an API for the app.

Which API should you use?

正确答案: C. SQL API

说明：

当应用需要处理关系型数据时，CosmosDB的SQL API是最合适的选择。SQL API是CosmosDB的原生API，支持SQL查询语法，能够很好地处理关系型数据。它提供了最丰富的功能集，包括存储过程、触发器和用户定义函数等。

其他选项分析：

A. MongoDB API - 适用于文档型数据，不是处理关系型数据的最佳选择

B. Table API - 主要用于简单的键值存储，不适合复杂的关系型数据

D. Cassandra API - 适用于宽列存储场景，不是处理关系型数据的最佳选择
