You are developing a website that will run as an Azure Web App. Users will authenticate by using their Azure Active Directory (Azure AD) credentials. You plan to assign users one of the following permission levels for the website: admin, normal, and reader. A user's Azure AD group membership must be used to determine the permission level. You need to configure authorization.

Solution: Configure the Azure Web App for the website to allow only authenticated requests and require Azure AD log on.

Does the solution meet the goal?

正确答案: B. No

说明:

该解决方案仅配置了身份验证（Authentication）而没有实现授权（Authorization）功能：

  • 问题点1：解决方案只要求Azure AD登录（身份验证），但没有配置如何根据Azure AD组 membership来分配不同的权限级别（admin/normal/reader）[4](@ref)
  • 问题点2：题目明确要求基于Azure AD组 membership实现权限分级，而解决方案仅实现了基础的身份验证功能[5](@ref)
  • 缺失部分：需要额外配置组声明(groupMembershipClaims)和应用角色分配，才能实现基于组的权限控制[5](@ref)[6](@ref)

要实现题目要求，还需要：

  1. 在应用清单中配置groupMembershipClaims为"All"以获取组信息
  2. 在代码中实现基于组 membership的权限检查逻辑
  3. 或在Azure AD中配置应用角色并通过组分配来实现权限控制[6](@ref)
