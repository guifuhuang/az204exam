QUESTION NO: 320

Your company is designing an application named App1 that will use data from Azure SQL Database. App1 will be accessed over the internet by many users.

You need to recommend a solution for improving the performance of App1.

What should you include in the recommendation?

正确答案: D. Azure Cache for Redis

说明:

1. 问题分析：题目描述的是一个需要从Azure SQL数据库获取数据并被大量用户访问的互联网应用，需要提升性能。

2. 为什么选择Redis：

  • Redis作为内存缓存可以显著减少数据库查询次数，提高响应速度
  • 支持缓存常用查询结果，减轻SQL数据库负载
  • 适用于高并发场景，能够处理大量用户请求
  • 提供数据分片和复制功能，可水平扩展

3. 其他选项分析：

  • A. Azure HPC缓存：用于高性能计算工作负载，不适合常规Web应用
  • B. ExpressRoute：提供专用网络连接，不直接提高应用性能
  • C. CDN：适用于静态内容分发，不适用于动态数据库查询

4. 典型应用场景：

Redis特别适合缓存数据库查询结果、会话状态和频繁访问的数据，能有效提升应用响应速度并降低数据库负载。
