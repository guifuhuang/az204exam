QUESTION NO: 341 HOTSPOT

You are developing an application that includes two Docker containers.

The application must meet the following requirements:

  • The containers must not run as root.
  • The containers must be deployed to Azure Container Instances by using a YAML file.
  • The containers must share a lifecycle, resources, local network and storage volume.
  • The storage volume must persist through container crashes.
  • The storage volume must be destroyed on stop or restart of the containers.

You need to configure Azure Container Instances for the application.

Configuration setting Configuration value
Shared lifecycle
Storage volume

正确答案:

Shared lifecycle: Container group

Storage volume: Azure file share

解释说明:

1. Container group 是实现共享生命周期的正确选择。容器组(Container group)是Azure容器实例中的顶级资源，可以包含多个容器实例，这些容器共享生命周期、资源、本地网络和存储卷。

2. Azure file share 是满足存储卷要求的正确选择。Azure文件共享提供持久化存储，在容器崩溃时保留数据，同时可以通过配置在容器组停止时自动删除，符合题目要求。

这些选择是基于以下原因：

  • 容器组是唯一支持多个容器共享资源的配置
  • Azure文件共享提供持久化存储，适合需要保留数据的场景
  • 通过配置可以在容器组停止时自动删除文件共享
  • 空目录(Empty directory)虽然会在容器崩溃时保留数据，但不如Azure文件共享稳定可靠
