QUESTION NO: 323

You develop and deploy an Azure App Service web app to a production environment. You enable the Always On setting and the Application Insights site extensions. You deploy a code update and receive multiple failed requests and exceptions in the web app.

You need to validate the performance and failure counts of the web app in near real time.

Which Application Insights tool should you use?

正确答案: D. Live Metrics Stream

说明:

这个答案是正确的。Live Metrics Stream（实时指标流）是Application Insights中用于实时监控应用程序性能的工具，它可以：

  • 在代码更新后立即显示失败的请求和异常
  • 提供近实时的性能数据（延迟仅约1秒）
  • 显示服务器请求率、失败请求数和实时异常
  • 允许在部署后立即验证应用程序健康状况

其他选项不适合这个场景：

  • A. Snapshot Debugger：用于调试生产环境中的异常，不是实时监控工具
  • B. Profiler：用于性能分析，不是实时监控
  • C. Smart Detection：自动检测异常模式，但不是实时的
  • E. Application Map：显示应用程序组件关系，不是实时监控工具
