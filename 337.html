QUESTION NO: 337 HOTSPOT

You are developing a solution to store documents in Azure Blob storage. Customers upload documents to multiple containers. Documents consist of PDF, CSV, Microsoft Office format, and plain text files.

The solution must process millions of documents across hundreds of containers. The solution must meet the following requirements:

  • Documents must be categorized by a customer identifier as they are uploaded to the storage account.
  • Allow filtering by the customer identifier.
  • Allow searching of information contained within a document.
  • Minimize costs.

You created and configured a standard general-purpose v2 storage account to support the solution.

You need to implement the solution.

NOTE: Each correct selection is worth one point.

Requirement Solution
Search and filter by customer identifier.
Search information inside documents.

正确答案:

Search and filter by customer identifier: Azure Blob index tags

Search information inside documents: Azure Cognitive Search

解释说明:

1. Azure Blob index tags 是用于按客户标识符进行搜索和筛选的最佳解决方案。Blob索引标签允许您为每个Blob存储资源添加键值对标签，这些标签可以用于高效地分类和筛选大量文档，同时成本较低。

2. Azure Cognitive Search 是在文档内部搜索信息的最佳选择。它可以提取和索引各种文档格式(PDF、CSV、Office文档等)中的内容，提供全文搜索功能，非常适合搜索文档内部信息的需求。

这些选择是基于以下原因：

  • Blob索引标签专门设计用于基于元数据的筛选和分类，成本低且效率高
  • Azure Cognitive Search是专门的内容搜索服务，支持多种文档格式的内容提取
  • 组合使用这两种服务可以在满足需求的同时最小化成本
  • 其他选项如Blob元数据或清单策略无法同时满足搜索和筛选的需求
