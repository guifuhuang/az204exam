测试题：

问题描述：您正在开发一个 Azure App Service Web 应用。该 Web 应用必须将会话信息安全地存储在 Azure Redis Cache 中。您需要将 Web 应用连接到 Azure Redis Cache。

问题：应该使用哪三个 Azure Redis Cache 属性？（每个正确答案得1分）

答案与解析：

正确答案：A, D, E

解析：

  • A. SSL 端口：用于建立加密连接，确保会话数据传输安全（默认端口6380）。
  • D. 主机名：Redis 实例的唯一地址（格式：<名称>.redis.cache.windows.net）。
  • E. 访问密钥：身份验证凭证，用于安全访问 Redis 实例。

错误选项说明：

  • B/F（订阅名称/ID）和 C（位置）：属于 Azure 资源管理属性，与连接 Redis 无关。
