QUESTION NO: 131

You develop a gateway solution for a public facing news API. The news API back end is implemented as a RESTful service and uses an OpenAPI specification.

You need to ensure that you can access the news API by using an Azure API Management service instance.

Which Azure PowerShell command should you run?

正确答案：

A. Import-AzureRmApiManagementApi –Context $ApiMgmtContext –SpecificationFormat "Swagger" -SpecificationPath $SwaggerPath –Path $Path

说明：

  • 选项A正确：
    • 题目要求通过API Management访问使用OpenAPI规范的RESTful服务
    • Import-AzureRmApiManagementApi命令专门用于导入API定义
    • "Swagger"是OpenAPI规范的前身，两者兼容
    • 此命令会创建API Management中的API代理，使前端可以通过API Management访问后端服务
  • 其他选项分析：
    • 选项B：New-AzureRmApiManagementBackend用于创建后端服务配置，但不会建立API代理
    • 选项C：New-AzureRmApiManagement用于创建API Management服务实例，不是题目要求的操作
    • 选项D：New-AzureRmApiManagementBackendProxy不是有效的PowerShell命令

题目核心要求是"access the news API by using an API Management service"，即建立API Management与后端API的连接，这正是Import-AzureRmApiManagementApi命令的功能。
