QUESTION NO: 319

You need to design network connectivity for a subnet in an Azure virtual network. The subnet will contain 30 virtual machines. The virtual machines will establish outbound connections to internet hosts by using the same a pool of four public IP addresses, inbound connections to the virtual machines will be prevented.

What should include in the design?







正确答案:

B. NAT Gateway

说明:

为什么选择NAT Gateway:

  • NAT网关专门用于提供出站互联网连接，同时阻止入站连接
  • 支持多个VM共享一组公共IP地址（题目要求的4个IP）
  • 无需在VM上配置公共IP地址即可实现出站连接
  • 自动阻止入站连接，符合安全要求
  • Azure推荐用于大规模出站连接场景

其他选项为什么不正确:

  • A. Azure Private Link: 用于安全访问Azure服务，不提供出站互联网连接
  • C. User Defined Routes: 用于自定义网络流量路由，无法实现IP地址共享
  • D. Azure Virtual WAN: 用于大规模网络连接中心，不解决具体出站连接需求

NAT Gateway关键特性:

  1. 支持最多16个公共IP地址的地址池
  2. 单个NAT网关可处理最多64,000个并发流
  3. 自动SNAT（源网络地址转换）
  4. 与网络安全组(NSG)配合实现完全的网络隔离

实现架构示例:

Virtual Machines (30) → Subnet → NAT Gateway (4 Public IPs) → Internet
            ↑
            └─────────────× 入站连接被自动阻止
