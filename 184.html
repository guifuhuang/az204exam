QUESTION NO: 184 HOTSPOT

You are configuring a development environment for your team. You deploy the latest Visual Studio image from the Azure Marketplace to your Azure subscription.

The development environment requires several software development kits (SDKs) and third-party components to support application development across the organization. You install and customize the deployed virtual machine (VM) for your development team. The customized VM must be saved to allow provisioning of a new team member development environment.

You need to save the customized VM for future provisioning.

Which tools or services should you use? To answer, select the appropriate options in the answer area.

NOTE: Each correct selection is worth one point.

Action Tool or service
Generalize the VM
Store images

正确答案:

Generalize the VM: Azure PowerShell

Store images: Azure Blob Storage

说明:

  1. 通用化虚拟机:
    • 需要使用Azure PowerShell中的Set-AzVm -Generalized命令或Azure CLI中的az vm generalize命令来通用化VM
    • 这是创建自定义镜像前的必要步骤，会移除所有机器特定信息
  2. 存储镜像:
    • Azure Blob Storage是存储VM镜像的标准服务
    • 通用化后，可以使用Save-AzVMImage命令将镜像保存到Blob存储
    • 存储的镜像可以用于后续创建相同配置的新VM

其他选项不适合此场景：Visual Studio命令提示符不用于VM管理，Azure Migrate用于迁移评估，Azure Backup用于备份而非镜像创建，其他存储服务不用于存储VM镜像。
