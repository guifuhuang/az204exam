QUESTION NO: 91 HOTSPOT

You are creating a CLI script that creates an Azure web app related services in Azure App Service. The web app uses the following variables:

Variable name Value
$gitrepo https://github.com/contoso/os/webapp
$webappname Webapp1103

You need to automatically deploy code from GitHub to the newly created web app.

How should you complete the script? To answer, select the appropriate options in the answer area.

NOTE: Each correct selection is worth one point.

# Create a resource group az group create --location westeurope --name myResourceGroup # Create an App Service plan az --name $webappname --resource-group myResourceGroup --sku # Create a web app az --name $webappname --resource-group myResourceGroup # Configure GitHub deployment az source config --name $webappname --resource-group myResourceGroup --repo-url $gitrepo --branch master --manual-integration

Correct Answers:

  • First dropdown: appservice plan create
  • SKU dropdown: FREE
  • Second dropdown: webapp create
  • Third dropdown: --plan $webappname
  • Fourth dropdown: webapp deployment

Explanation:

The complete script should be:

# Create a resource group az group create --location westeurope --name myResourceGroup # Create an App Service plan az appservice plan create --name $webappname --resource-group myResourceGroup --sku FREE # Create a web app az webapp create --name $webappname --resource-group myResourceGroup --plan $webappname # Configure GitHub deployment az webapp deployment source config --name $webappname --resource-group myResourceGroup --repo-url $gitrepo --branch master --manual-integration

Key points:

  1. Use appservice plan create to create the App Service plan
  2. The FREE SKU is appropriate for this scenario
  3. Use webapp deployment source config to configure GitHub deployment
  4. The --manual-integration flag means you'll need to manually sync changes
