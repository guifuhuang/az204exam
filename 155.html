QUESTION NO: 155

You develop an Azure web app. You monitor performance of the web app by using Application Insights.

You need to ensure the cost for Application Insights does not exceed a preset budget.

What should you do?









正确答案：

B. Set a daily cap for the Application Insights instance.

说明：

  • 选项B正确：
    • 题目要求确保成本不超过预设预算，最直接的方法是设置每日上限
    • 每日上限是Application Insights原生的成本控制功能
    • 当数据量达到上限时会停止数据收集，直到次日重置
    • 可以在Azure门户中配置："Application Insights资源" → "使用和预估成本" → "每日上限"
  • 其他选项分析：
    • 选项A/C/D/E：采样技术可以降低数据量但不能保证绝对不超预算
    • 采样技术更适合减少数据量同时保持代表性，而非严格的成本控制
    • 自适应采样会根据流量自动调整采样率，但仍无法保证绝对不超预算

根据Microsoft官方文档，设置每日上限是确保成本不超过预算的最可靠方法。采样技术应作为补充措施使用。
