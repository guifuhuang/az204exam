QUESTION NO: 126 DRAG DROP

You are deploying an Azure Kubernetes Services (AKS) cluster that will use multiple containers.

You need to create the cluster and verify that the services for the containers are configured correctly and available.

Which four commands should you use to develop the solution? To answer, move the appropriate command segments from the list of command segments to the answer area and arrange them in the correct order.

Command segments

az group create
az appservice plan create
az aks create
az aks get-credentials
kubectl apply

Answer Area

⇦⇦
⇩⇩

正确答案:

  1. az group create
  2. az aks create
  3. az aks get-credentials
  4. kubectl apply

说明:

创建AKS集群的正确命令顺序应该是：

  1. 首先创建资源组 (az group create)
  2. 然后创建AKS集群 (az aks create)
  3. 获取集群凭证 (az aks get-credentials)
  4. 最后部署应用 (kubectl apply)

注意：图片中显示的"az appservice plan create"命令不适用于AKS集群部署，这是用于Azure App Service的命令。
