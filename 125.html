QUESTION NO: 125

You are developing a software solution for an autonomous transportation system. The solution uses large data sets and Azure Batch processing to simulate navigation sets for entire fleets of vehicles.

You need to create compute nodes for the solution on Azure Batch.

What should you do?

正确答案:

D. In a .NET method, call the method: BatchClient.PoolOperations.CreateJob

解析说明:

在Azure Batch中创建计算节点的正确流程是：

  1. 创建计算节点池(Pool) - 指定节点数量、大小和操作系统
  2. 创建工作(Job) - 使用BatchClient.PoolOperations.CreateJob方法（选项D正确）
  3. 添加任务(Task) - 每个任务执行应用程序处理数据

关键说明:

Batch作业(Job)是一个或多个任务的逻辑分组。作业包含任务共有的设置，如优先级和运行任务的池。应用程序使用BatchClient.JobOperations.CreateJob方法在池上创建作业。

各选项分析:

  • 选项A：TaskAddParameter用于定义任务，不是创建计算节点
  • 选项B：JobAddParameter是Python中的作业创建类，但题目明确要求使用.NET方法
  • 选项C：创建Batch账户是前提条件，但不是直接创建计算节点
  • 选项D：CreateJob方法用于在指定池上创建作业，是创建计算节点工作流的关键步骤
