QUESTION NO: 427 HOTSPOT

You are developing several microservices to run on Azure Container Apps.

You need to monitor and diagnose the microservices.

Which features should you use? To answer, select the appropriate feature in the answer area.

NOTE: Each correct selection is worth one point.

Answer Area Requirement Feature
View console logs from a container in near-real-time
Debug the microservice from inside the container

正确答案:

Requirement Correct Feature Technical Explanation
View console logs from a container in near-real-time Log streaming Provides real-time streaming of container logs
Debug the microservice from inside the container Container console Allows direct access to container for debugging

技术说明:

功能对比:

  • Log streaming: 实时查看容器日志，适合监控应用输出
  • Container console: 提供容器内交互式shell，用于调试
  • Azure Monitor Log Analytics: 用于历史日志分析和查询
  • Azure Container Registry: 容器镜像存储，不直接用于监控

Azure Container Apps 监控架构:

1. Log Streaming → 通过Azure门户或CLI实时查看日志

2. Container Console → 使用exec命令进入运行中的容器

3. 诊断流程: 实时日志 → 控制台调试 → 日志分析

生产环境建议:

  1. 开发阶段使用Log streaming和Container console进行调试
  2. 生产环境配置Azure Monitor进行长期日志存储和分析
  3. 结合Application Insights实现端到端监控
