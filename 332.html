QUESTION NO: 332

You are building a web application that performs image analysis on user photos and returns metadata containing objects identified. The image analysis is very costly in terms of time and compute resources. You are planning to use Azure Redis Cache so Cache uploads do not need to be reprocessed.

In case of an Azure data center outage metadata loss must be kept to a minimum.

You need to configure the Azure Redis cache instance.

Which two actions should you perform? Each correct answer presents part of the solution.

NOTE: Each correct selection is worth one point.

正确答案: A, C

说明：

1. 配置Azure Redis持久化(A)是必要的，这样可以确保数据在数据中心故障时不会丢失。文档中提到"数据持久性允许你将Redis中存储的数据持久化"。

2. 将备份频率设置为最小值(C)可以最大限度地减少数据丢失，因为文档指出"你可以拍摄快照或写入仅附加文件来备份数据"。

选项B不正确，因为不需要配置第二个存储账户；选项D不正确，因为RDS持久化不是Azure Redis的有效配置选项。
