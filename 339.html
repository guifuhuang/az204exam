QUESTION NO: 339 HOTSPOT

You are developing a service where customers can report news events from a browser using Azure Web PubSub. The service is implemented as an Azure App that uses the JSON WebSocket subprotocol to receive news events.

You need to implement the bindings for the Azure Function App.

How should you configure the binding? To answer, select the appropriate options in the answer area.

Note: Each correct selection is worth one point.

{
  "bindings": [
    {
      "type": "
                
              ",
      "direction": "in",
      "name": "data",
      "eventName": "message",
      "eventType": "
                     
                   "
    }
  ]
}

正确答案:

type: system

eventType: message

解释说明:

1. system 是用于Azure Web PubSub绑定的正确类型。当需要处理来自Web PubSub服务的系统事件时，应该使用这种类型。

2. message 是正确的事件类型。因为题目描述中提到服务需要接收新闻事件(message events)，所以应该将eventType设置为"message"。

这些选择是基于以下原因：

  • "system"类型用于处理Web PubSub服务的系统级事件
  • "message"事件类型对应WebSocket协议中的消息事件
  • 题目描述中明确提到使用JSON WebSocket子协议接收新闻事件
  • 配置中的"eventName": "message"也提示了应该使用"message"作为eventType
