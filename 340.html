QUESTION NO: 340 HOTSPOT

You are developing an Azure Static Web app that contains training materials for a tool company. Each tool's training material is contained in a static web page that is linked from the tool's publicly available description page.

A user must be authenticated using Azure AD prior to viewing training.

You need to ensure that the user can view training material pages after authentication.

正确答案:

routes: routes

redirect: %auth.login

auth provider: aad

post login: post_login.microsoftonline

解释说明:

1. routes 是处理认证和重定向的正确配置部分。它定义了不同HTTP状态码的处理方式。

2. %auth.login 是404错误的正确重定向目标。当用户尝试访问需要认证的资源时，会被重定向到登录页面。

3. aad 是Azure Active Directory的身份提供者标识。题目要求使用Azure AD进行认证。

4. post_login.microsoftonline 是Azure AD登录后的正确回调端点。这是Microsoft在线服务的标准登录后端点。

这些选择是基于以下原因：

  • routes配置处理认证流程和错误页面
  • %auth.login是静态Web应用的标准登录重定向变量
  • aad是Azure Active Directory的官方标识符
  • post_login.microsoftonline是Azure AD的标准登录后端点
