QUESTION NO: 316

You are designing a small app that will receive web requests containing encoded geographic coordinates. Calls to the app will occur infrequently.

Which compute solution should you recommend?

正确答案: B. Azure App Service

说明:

1. Azure App Service 是最适合此场景的解决方案，因为：

  • 题目描述的是一个需要接收HTTP请求的小型应用，App Service专为Web应用设计
  • 调用频率低（infrequent）意味着不需要高扩展性，App Service的基础层即可满足
  • 相比无服务器方案（如Functions），App Service提供更简单的部署和管理方式

2. 其他选项分析：

  • A. Azure Functions：虽然可以处理HTTP请求，但更适合事件驱动、短时间运行的场景
  • C. Azure Batch：用于大规模并行计算任务，不适合简单的Web请求处理
  • D. Azure API Management：是API网关而非计算服务，不能直接处理请求

3. 成本考虑：

App Service的基础层（如B1）成本低廉，适合低频访问的小型应用，同时提供完整的Web托管能力。
