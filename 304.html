QUESTION NO: 304

You are developing an Azure Function App that runs in an App Service Plan. The Azure Function is triggered by a Timer object.

You observe that the Azure Function does not reliably trigger when scheduled.

Which two actions should you perform?

正确答案:

A. Verify that Always On is enabled.

D. Modify the trigger to use Consumption mode instead of the App Service plan.

说明:

1. 对于在App Service Plan中运行的Azure Functions，需要启用"Always On"设置以确保定时触发器可靠执行。因为App Service Plan在空闲时会卸载应用以节省资源，导致定时触发器可能无法按时触发。

2. 另一个解决方案是将函数改为使用消费计划(Consumption mode)，因为消费计划会自动管理函数的执行环境，确保定时触发器可靠触发。

选项B改为SignalR触发器不相关，选项C配置重试对于定时触发器不可靠的问题没有帮助。
