QUESTION NO: 318

Your company purchases an Azure subscription and plans to migrate several on-premises virtual machines to Azure. You need to design the infrastructure required for the Azure virtual machines solution.

What should you include in the design? (Select all that apply)







正确答案:

C. the size of the virtual machines

说明:

为什么选择C:

  • 根据搜索结果[1][6][7]，VM大小是迁移评估的核心要素：
    • Azure Migrate评估工具会基于CPU/内存使用率推荐VM尺寸
    • 直接影响计算性能和成本优化
    • 必须匹配本地工作负载需求

其他选项为什么不正确:

  • A. 存储账户数量：
    • 现代架构使用托管磁盘，存储账户由平台自动管理
  • B. 虚拟网络设置：
    • 虽然是重要组件，但题目明确要求"infrastructure required for VMs"
    • 网络属于共享平台服务，非VM专属配置
  • D. 区域数量：
    • 初始迁移通常只需单区域
    • 多区域属于高可用性设计，非基本需求

关键设计流程:

  1. 使用Azure Migrate评估本地VM性能数据
  2. 根据CPU/内存使用率确定目标VM尺寸
  3. 选择磁盘类型（Standard HDD/Premium SSD）
  4. 验证Azure订阅配额是否支持所选VM系列
