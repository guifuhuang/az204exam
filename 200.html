QUESTION NO: 200

A company is developing a solution that allows smart refrigerators to send temperature information to a central location. You have an existing Service Bus.

The solution must receive and store messages until they can be processed. You create an Azure Service Bus instance by providing a name, pricing tier, subscription, resource group, and location.

You need to complete the configuration.

Which Azure CLI or PowerShell command should you run?

正确答案: B. Option B

说明:

题目要求完成Service Bus的配置，以接收和存储智能冰箱的温度信息消息。根据题目描述和提供的选项：

  1. 选项分析:
    • A选项：创建Service Bus命名空间，但题目中已说明"已有Service Bus"并创建了实例
    • B选项：创建Service Bus队列，这是正确答案，因为队列正是用于接收和存储消息的组件
    • C选项：获取连接字符串，这是后续使用的步骤，不是当前需要的配置
    • D选项：创建资源组，题目中已提供资源组信息
  2. 为什么选择B:
    • 题目明确要求"接收和存储消息"，这正是Service Bus队列的功能
    • 虽然已有Service Bus实例，但需要创建队列来具体处理消息
    • 队列是消息存储的物理容器，必须显式创建
  3. 命令详解:
    az servicebus queue create --resource-group fridge-rg # 指定资源组 --namespace-name fridge-ns # 指定Service Bus命名空间 --name fridge-q # 指定队列名称

关键点:

  • Service Bus命名空间是消息服务的高级容器
  • 队列是实际存储消息的实体
  • 题目中已完成命名空间创建，下一步需要创建具体队列
  • 连接字符串和资源组创建都不是当前配置的核心需求
