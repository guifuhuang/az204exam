QUESTION NO: 265

You are developing an Azure App Service REST API.

The API must be called by an Azure App Service web app. The API must retrieve and update user profile information stored in Azure Active Directory (Azure AD).

You need to configure the API to make the updates.

Which two tools should you use? Each correct answer presents part of the solution.

NOTE: Each correct selection is worth one point.

正确答案: A. Microsoft Graph API 和 C. Azure API Management

说明:

1. Microsoft Graph API 是访问和更新Azure AD中用户信息的正确方式。企业开发者使用Microsoft Graph在Azure AD资源和其他服务之间创建独特的工作流，实现身份管理和自动化管理流程，如员工入职(和离职)、个人资料维护、许可证部署等。

2. Azure API Management (APIM) 是为现有后端服务创建一致且现代化的API网关的方式。API Management帮助组织向外部、合作伙伴和内部开发者发布API，释放其数据和服务的潜力。

其他选项说明：

- B. MSAL虽然用于身份验证，但不是此场景的最佳选择

- D. Azure Security Center是安全监控工具，不相关

- E. Azure Key Vault SDK用于管理密钥和机密，不用于用户数据操作
