QUESTION NO: 241 DRAG DROP

Question:

A company has multiple warehouses. Each warehouse contains IoT temperature devices which deliver temperature data to an Azure Service Bus queue.

You need to send email alerts to facility supervisors immediately if the temperature at a warehouse goes above or below specified threshold temperatures.

Which five actions should you perform in sequence? To answer, move the appropriate actions from the list of actions to the answer area and arrange them in the correct order.

Actions

Add a Logic Apps trigger that fires when one or more messages arrive in the queue.
Add a Recurrence trigger that configures the app to run every 15 minutes.
Add an action to send an email to specified personnel if the temperature is outside the set thresholds.
Add a trigger to read IoT temperature data from the Service Bus queue.
Add a Logic Apps action that fires when one or more messages arrive in the queue.
Add a condition to compare the temperature against upper and lower thresholds.
Create a blank Logic App.
Add an action to read IoT temperature data from the Service Bus queue.

Answer Area

Drag and drop the correct actions here in the right order

Correct Answer and Explanation

Correct Sequence:

  1. Create a blank Logic App.
  2. Add a Logic Apps trigger that fires when one or more messages arrive in the queue.
  3. Add an action that reads IoT temperature data from the Service Bus queue.
  4. Add a condition to compare the temperature against upper and lower thresholds.
  5. Add an action to send an email to specified personnel if the temperature is outside the set thresholds.

解析：

1. 创建一个空白逻辑应用：这是所有工作的起点，需要先创建Logic App资源。

2. 添加队列消息触发器：配置当Service Bus队列收到消息时触发逻辑应用，这是实时处理的关键。

3. 添加读取温度数据的操作：从Service Bus队列中获取IoT设备发送的温度数据。

4. 添加条件判断：将读取到的温度值与预设的上下阈值进行比较。

5. 添加发送邮件操作：当温度超出阈值时，触发邮件发送给指定的设施主管。

注意：Recurrence触发器（每15分钟运行一次）不符合题目要求的"立即"响应，因此不应选择。
