Margie's Travel is an international travel and bookings management service. The company is expanding into restaurant bookings. You are tasked with implementing Azure Search for the restaurants listed in their solution.

You create the index in Azure Search.

You need to import the restaurant data into the Azure Search service by using the Azure Search .NET SDK.

Solution:

1. Create a SearchIndexClient object to connect to the search index.

2. Create an IndexBatch that contains the documents which must be added.

3. Call the Documents.Index method of the SearchIndexClient and pass the IndexBatch.

Does the solution meet the goal?

正确答案:

A. Yes

解析说明:

根据Azure Search .NET SDK的正确使用方式：

  1. 解决方案正确地使用了SearchIndexClient连接到搜索索引
  2. 创建IndexBatch是上传文档的标准方式
  3. 调用Documents.Index方法并传递IndexBatch是上传文档的正确API调用

代码验证:

// 正确使用SearchIndexClient上传文档的示例代码
var searchClient = new SearchIndexClient(
    new Uri("https://[service-name].search.windows.net"),
    new AzureKeyCredential("[admin-key]"));

// 创建文档批次
var batch = IndexBatch.Upload(documents);

// 上传文档
await searchClient.Documents.IndexAsync(batch);

题目中的解决方案完全符合Azure Search .NET SDK的正确使用方式，能够实现将餐厅数据导入Azure Search的目标。
