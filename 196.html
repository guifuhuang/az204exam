QUESTION NO: 196 HOTSPOT

You plan to deploy a web app to App Service on Linux. You create an App Service plan. You create and push a custom Docker image that contains the web app to Azure Container Registry.

You need to access the console logs generated from inside the container in real-time.

How should you complete the Azure CLI command? To answer, select the appropriate options in the answer area.

NOTE: Each correct selection is worth one point.

az webapp log --name ContosoWeb --resource-group ContosoDevRG filesystem
az log --name ContosoWeb --resource-group ContosoDevRG

正确答案：

az webapp log tail --name ContosoWeb --resource-group ContosoDevRG --docker-container-logging filesystem
az webapp log download --name ContosoWeb --resource-group ContosoDevRG

说明：

  1. 第一行命令使用 tail 子命令来实时流式传输日志
  2. 第二行参数组合为：
    • --docker-container-logging 用于查看Docker容器内的日志

关键点解释：

  • tail 是唯一支持实时日志流的子命令
  • docker-container-logging 参数专门用于容器日志（而不是文件系统或应用日志）
  • 完整的有效命令是：
    az webapp log tail --name ContosoWeb --resource-group ContosoDevRG --docker-container-logging filesystem
    az webapp log download --name ContosoWeb --resource-group ContosoDevRG
