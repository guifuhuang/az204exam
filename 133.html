QUESTION NO: 133

You are creating a hazard notification system that has a single signaling server which triggers audio and visual alarms to start and stop.

You implement Azure Service Bus to publish alarms. Each alarm controller uses Azure Service Bus to receive alarm signals as part of a transaction. Alarm events must be recorded for audit purposes. Each transaction record must include information about the alarm type that was activated.

You need to implement a reply trail auditing solution.

Which two actions should you perform? Each correct answer presents part of the solution.

NOTE: Each correct selection is worth one point.

正确答案:

A. Assign the value of the hazard message SessionID property to the ReplyToSessionId property.

D. Assign the value of the hazard message MessageId property to the CorrelationId property.

解析说明:

要实现审计追踪解决方案，需要正确设置以下两个属性：

选项 解释
A (正确) 将SessionID赋值给ReplyToSessionId可以确保回复消息与原始消息关联，这对于审计追踪至关重要
D (正确) 将MessageId赋值给CorrelationId可以建立消息之间的关联关系，便于追踪整个事务流程
B (错误) DeliveryCount是系统自动维护的属性，不应手动设置
C (错误) SequenceNumber是系统分配的序号，不应手动设置
E (错误) SequenceNumber和DeliveryCount是完全不同的属性，不应互相赋值
F (错误) MessageId不应赋值给SequenceNumber，这会破坏消息的顺序追踪

关键点分析:

  • ReplyToSessionId：用于确保回复消息能正确关联到原始会话，这对审计追踪至关重要
  • CorrelationId：用于在分布式系统中追踪相关消息，是审计追踪的核心属性
  • 系统自动维护的属性（如SequenceNumber、DeliveryCount）不应手动修改
  • 审计追踪需要完整的消息关联关系，而不仅仅是消息内容

Azure Service Bus 审计最佳实践:

  1. 使用CorrelationId建立消息间的关联关系
  2. 使用SessionId和ReplyToSessionId确保消息会话完整性
  3. 避免修改系统自动维护的属性
  4. 考虑启用消息的重复检测功能以提高可靠性
