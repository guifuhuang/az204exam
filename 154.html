QUESTION NO: 154

You develop a website. You plan to host the website in Azure. You expect the website to experience high traffic volumes after it is published. You must ensure that the website remains available and responsive while minimizing cost. You need to deploy the website. What should you do?

正确答案: C

C. Deploy the website to an App Service that uses the Standard service tier. Configure the App Service plan to automatically scale when the CPU load is high.

解析:

根据题目要求和Azure最佳实践：

  1. App Service vs Virtual Machine：对于高流量网站，Azure App Service是更好的选择，因为它提供内置的自动扩展、负载均衡和高可用性功能，而虚拟机需要更多的手动配置和管理[2](@ref)[5](@ref)。
  2. 服务层级选择：Shared服务层级不适合高流量场景，因为它有严格的资源限制且不支持自动扩展。Standard服务层级提供更好的性能和自动扩展能力，同时保持成本效益[5](@ref)[6](@ref)。
  3. 自动扩展配置：在Standard服务层级上配置基于CPU负载的自动扩展可以确保网站在高流量时保持响应，同时在流量降低时自动缩减资源以节省成本[2](@ref)[5](@ref)。
  4. 成本优化：相比虚拟机规模集（选项D），App Service的Standard层级提供了更好的性价比，因为它消除了管理基础设施的开销，同时提供类似的扩展能力[5](@ref)[6](@ref)。

其他选项分析：

  • 选项A：Shared服务层级不支持自动扩展，不适合高流量场景
  • 选项B：单个虚拟机无法自动扩展，且管理成本高
  • 选项D：虽然虚拟机规模集支持扩展，但相比App Service需要更多管理且成本更高
