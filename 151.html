QUESTION NO: 151 HOTSPOT

A company runs an international travel and bookings management service. The company plans to begin offering restaurant bookings. You must develop a solution that uses Azure Search and meets the following requirements:

  • Users must be able to search for restaurants by name, description, location, and cuisine.
  • Users must be able to narrow the results further by location, cuisine, rating, and family-friendliness.
  • All words in descriptions must be included in searches.

You need to add annotations to the restaurant class.

How should you complete the code segment? To answer, select the appropriate options in the answer area.

NOTE: Each correct selection is worth one point.

[SerializePropertyNameAsCamelCase] public class Restaurant { [Key, IsFilterable] public int RestaurantId { get; set; } [IsSearchable, IsFilterable, IsSortable] public string Name { get; set; }
public string location { get; set; } public string Phone { get; set; }
public string Description { get; set; }
public double Rating { get; set; }
public List<string> Cuisines { get; set; }
public bool FamilyFriendly { get; set; } }

正确答案:

location: [IsSearchable, IsFilterable, IsSortable, IsFacetable]

Description: [IsSearchable]

Rating: [IsFilterable, IsSortable, IsFacetable]

Cuisines: [IsSearchable, IsFilterable, IsFacetable]

FamilyFriendly: [IsFilterable, IsFacetable]

解析:

根据题目要求：

  1. location属性需要能被搜索(name, description, location, cuisine)和进一步筛选(location, cuisine, rating, family-friendliness)，因此需要[IsSearchable, IsFilterable, IsSortable, IsFacetable]
  2. Description属性需要包含所有单词在搜索中，只需要[IsSearchable]
  3. Rating属性需要能被筛选和排序，因此需要[IsFilterable, IsSortable, IsFacetable]
  4. Cuisines属性需要能被搜索和筛选，因此需要[IsSearchable, IsFilterable, IsFacetable]
  5. FamilyFriendly属性需要能被筛选，因此需要[IsFilterable, IsFacetable]
