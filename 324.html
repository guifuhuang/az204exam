QUESTION NO: 324

You are building a web application that uses the Microsoft identity platform for user authentication.

You are implementing user identification for the web application. You need to retrieve a claim to uniquely identify a user.

Which claim type should you use?

正确答案: A. oid

说明:

1. 问题分析：题目要求从Microsoft身份平台获取唯一标识用户的声明(claim)。

2. 为什么选择oid：

  • oid (Object ID) 是Azure AD中用户的全局唯一标识符(GUID)
  • 即使用户更改了用户名或邮箱，oid也保持不变
  • 在租户内每个用户的oid都是唯一的
  • 是Microsoft身份平台推荐的用户唯一标识方式

3. 其他选项分析：

  • B. aud：受众声明，表示令牌的目标接收者(通常是应用ID)，不是用户标识
  • C. idp：身份提供者声明，表示用户认证的来源，不能唯一标识用户
  • D. nonce：用于防止重放攻击的一次性值，与用户标识无关

4. 实际应用：

在代码中可以通过解析JWT令牌获取oid声明，例如：

var oid = User.FindFirst("http://schemas.microsoft.com/identity/claims/objectidentifier")?.Value;
