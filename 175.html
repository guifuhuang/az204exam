QUESTION NO: 175 DRAG DROP-Azure Application Insights

You develop an ASP.NET Core MVC application. You configure the application to track webpages and custom events. You need to identify trends in application usage.

Which Azure Application Insights Usage Analysis features should you use? To answer, drag the appropriate features to the correct requirements. Each feature may be used once, more than once, or not at all.

Features

Users
Impact
Retention
User flows

Requirements

1. Which pages visited by users most often correlate to a product purchase?

2. How does load time of the product display page affect a user's decision to purchase a product?

3. Which events most influence a user's decision to continue to use the application?

4. Are there places in the application that users often perform repetitive actions?

Correct Answers and Explanations:

1. Which pages visited by users most often correlate to a product purchase?

Answer: User flows

Explanation: User flows visualize the paths users take through your application, showing which pages are most frequently visited before a purchase. This helps identify conversion paths and drop-off points[4](@ref)[8](@ref).

2. How does load time of the product display page affect a user's decision to purchase a product?

Answer: Impact

Explanation: Impact analysis correlates technical metrics (like page load time) with business outcomes (purchases) to show how performance affects user behavior[6](@ref)[4](@ref).

3. Which events most influence a user's decision to continue to use the application?

Answer: Retention

Explanation: Retention analysis identifies which features or events cause users to return, showing what drives continued usage[4](@ref)[6](@ref)[9](@ref).

4. Are there places in the application that users often perform repetitive actions?

Answer: User flows

Explanation: User flows reveal patterns in navigation, including repetitive actions or loops in user behavior[4](@ref)[8](@ref).

正确答案与中文说明：

1. 用户最常访问哪些页面与产品购买相关？

正确答案：用户流

说明：用户流功能可以可视化用户通过应用程序的路径，显示哪些页面最常被访问后发生购买行为。这有助于识别转化路径和流失点[2](@ref)[6](@ref)。

2. 产品展示页面的加载时间如何影响用户购买产品的决定？

正确答案：影响分析

说明：影响分析将技术指标(如页面加载时间)与业务结果(如购买率)相关联，显示性能如何影响用户行为。这可以帮助团队就优化优先级做出决策[3](@ref)[6](@ref)。

3. 哪些事件最能影响用户决定继续使用应用程序？

正确答案：保留率

说明：保留率分析识别哪些功能或事件会导致用户返回，显示什么驱动了持续使用。例如，可以比较游戏胜利和失败用户的返回率差异[3](@ref)[7](@ref)。

4. 应用程序中是否有用户经常执行重复操作的地方？

正确答案：用户流

说明：用户流可以揭示导航模式，包括重复操作或用户行为中的循环，帮助识别用户可能遇到困难或重复执行相同操作的地方[2](@ref)[6](@ref)。
