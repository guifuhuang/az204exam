QUESTION NO: 288

An organization hosts web apps in Azure. The organization uses Azure Monitor. You discover that configuration changes were made to some of the web apps. You need to identify the configuration changes. Which Azure Monitor log should you review?

答案及说明：

正确答案：C. AppServiceAuditLogs

说明：

要识别Azure Web应用的配置更改，应该查看AppServiceAuditLogs（应用服务审核日志）。

各选项说明：

  • A. AppServiceEnvironmentPlatformLogs - 记录应用服务环境的平台日志，不是用于跟踪配置更改
  • B. AppServiceApplogs - 记录应用程序日志，不是配置更改日志
  • C. AppServiceAuditLogs - 正确选项，专门记录对应用服务的配置更改操作
  • D. AppServiceConsoteLogs - 此选项名称可能有误，正确的应该是AppServiceConsoleLogs（控制台日志），用于记录应用控制台输出

审核日志(AppServiceAuditLogs)会记录所有管理操作，包括创建、修改或删除Web应用及其设置的更改，是识别配置更改的最佳选择。
