Topic 7: 公司网站配置解决方案

公司网站要求

VanArsdel Ltd.需要为其公司网站选择合适的Azure服务配置。

具体要求：

  • 需要支持动态内容和企业级功能
  • 必须支持自定义域名和SSL证书
  • 需要自动扩展以应对流量高峰
  • 应支持持续部署和开发运维实践
  • 需要与Azure Active Directory集成进行身份验证

当前环境：

  • 公司网站使用React前端和Node.js后端
  • 需要支持用户认证和个性化内容
  • 预计每日访问量在10,000-50,000用户之间
  • 计划在未来6个月内增加电子商务功能

正确答案及说明

正确答案：Plan: Standard | Service: App Service Web App

✓ Plan: Standard - 正确
✓ Service: App Service Web App - 正确

详细说明：

为什么选择Standard计划和App Service Web App？

Standard计划优势

  • 支持自定义域名和SSL证书
  • 提供自动扩展功能应对流量高峰
  • 包含生产环境SLA(99.95%)
  • 支持部署槽位（staging slots）
  • 提供每日备份功能

App Service Web App优势

  • 完全支持动态内容和服务器端渲染
  • 内置支持Node.js、.NET、Python等
  • 与Azure DevOps无缝集成
  • 支持身份验证（Azure AD集成）
  • 提供高级监控和诊断功能

其他选项分析：

选项 适用场景 是否适合
Free计划 仅适用于开发和测试环境 ❌ 不适合生产环境
Premium计划 需要更高性能和隔离的场景 ⚠️ 过度配置，成本过高
Isolated计划 需要完全网络隔离的高安全性应用 ⚠️ 公司网站不需要此级别隔离
Static Web App 纯静态网站（如博客、宣传页） ❌ 不支持动态内容
Function App 事件驱动、无服务器架构 ⚠️ 不适合完整网站应用
Blob Storage 静态网站托管（仅HTML/CSS/JS） ❌ 不支持服务器端处理

实施建议：

  1. 在Azure门户中创建App Service Web App
  2. 选择Standard服务计划（S1或S2层级）
  3. 配置自定义域名和SSL证书
  4. 设置部署槽位用于蓝绿部署
  5. 配置自动扩展规则（基于CPU使用率或请求数）
  6. 启用Application Insights进行监控
  7. 设置每日自动备份

成本效益分析：

Standard计划提供最佳性价比：

  • 开发/测试环境：Free或Basic计划（$0-$55/月）
  • 生产环境（推荐）：Standard计划（$73-$300/月）
  • 高流量/高性能：Premium计划（$250-$1500+/月）
