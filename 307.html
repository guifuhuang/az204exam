QUESTION NO: 307

You are developing an application to manage shipping information for cargo ships. The application will use Azure Cosmos DB for storage.

The application must run offline when ships are at sea. The application must be connected to Azure when ships are in port.

Which Azure Cosmos DB API should you use for the application?

正确答案：

C. Cassandra

技术说明：

1. Cassandra API 最适合此场景的原因：

  • Cassandra数据库设计时就考虑了分布式和离线场景
  • 支持多主复制，非常适合需要在不同网络状态下工作的应用
  • 当船舶在海上离线时，可以继续本地操作
  • 靠港重新连接后，会自动同步数据变更
  • 数据冲突解决机制完善

2. 其他API的限制：

  • Core (SQL)：虽然支持离线，但不如Cassandra在分布式场景下表现好
  • MongoDB：离线支持有限，同步机制不如Cassandra完善
  • Gremlin：专为图数据库设计，不适合此业务场景

3. Cassandra解决方案优势：

  • 为航运业的特殊需求（频繁离线）提供最佳支持
  • 数据同步机制可靠，确保信息不丢失
  • 高性能写入，适合记录大量航运数据
  • 成熟的冲突解决策略
