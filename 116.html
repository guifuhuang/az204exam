QUESTION NO: 116

You are developing a solution that will use Azure messaging services.

You need to ensure that the solution uses a publish-subscribe model and eliminates the need for constant polling.

What are two possible ways to achieve the goal? Each correct answer presents a complete solution.

NOTE: Each correct selection is worth one point.

A. Service Bus
B. Event Hub
C. Event Grid
D. Queue

正确答案：

A. Service Bus

C. Event Grid

说明：

  • Service Bus (A)：Azure Service Bus 支持发布-订阅模型（通过主题和订阅），并采用推送机制而非轮询，完全符合题目要求。
  • Event Grid (C)：Azure Event Grid 是专门的事件路由服务，采用发布-订阅模型，并基于推送机制，无需轮询。
  • 排除选项 B (Event Hub)：虽然 Event Hub 可以处理大量事件，但它主要是数据摄取服务，不是典型的发布-订阅模型。
  • 排除选项 D (Queue)：队列是点对点通信模型，不支持发布-订阅模式。

题目要求同时满足两个条件：1) 发布-订阅模型；2) 消除轮询。只有 Service Bus 和 Event Grid 同时满足这两个要求。
