QUESTION NO: 243 HOTSPOT

Question:

You develop a news and blog content app for Windows devices.

A notification must arrive on a user's device when there is a new article available for them to view.

You need to implement push notifications.

How should you complete the code segment? To answer, select the appropriate options in the answer area.

NOTE: Each correct selection is worth one point.

string notificationHubName = "contoso_hub";
string notificationHubConnectionString = "connection_string";
hub = .(notificationHubConnectionString, notificationHubName);
string windowsToastPayload =
@"" +
@"New item to view" +@";"
try
{
var result = await hub.(windowsToastPayload);
...
}
catch (System.Exception ex)
{
// Handle exception
}

Correct Answer and Explanation

Correct Selections:

  1. NotificationHubClient (first dropdown)
  2. NotificationHubClient (second dropdown)
  3. CreateClientFromConnectionString
  4. SendWindowsNativeNotificationAsync

中文说明：

1. 第一个下拉框选择 NotificationHubClient: 这是声明hub变量的正确类型。

2. 第二个下拉框选择 NotificationHubClient: 这是调用静态方法的正确类名。

3. CreateClientFromConnectionString: 这是NotificationHubClient类中用于创建实例的正确静态方法。

4. SendWindowsNativeNotificationAsync: 这是向Windows设备发送原生通知的正确异步方法。

技术说明：

  • NotificationHubClient是Azure通知中心的主要客户端类
  • CreateClientFromConnectionString方法需要连接字符串和中心名称作为参数
  • SendWindowsNativeNotificationAsync专门用于发送Windows原生通知
  • 整个流程使用try-catch块来处理可能出现的异常

工作流程：

  1. 配置通知中心名称和连接字符串
  2. 创建NotificationHubClient实例
  3. 构建Windows Toast通知负载
  4. 使用try-catch块发送通知并处理异常
